feat(ffmpeg): Implement comprehensive Windows integration for FFmpeg (PR-ELECTRON-003)

Implements critical Windows-specific enhancements for FFmpeg integration in the Electron desktop app.
All requirements from PR-ELECTRON-003 have been completed and tested.

## Key Features

### 1. Windows Registry Detection
- Added comprehensive Windows Registry scanning for FFmpeg installations
- Searches HKLM and HKCU registry hives (both 64-bit and 32-bit views)
- Checks multiple registry paths and value names
- Falls back gracefully to PATH and common directories

### 2. Enhanced Path Detection
- Added user-specific installation paths (%LOCALAPPDATA%, %APPDATA%)
- Expanded common Windows installation locations
- Improved directory structure handling (bin subdirectories)

### 3. Windows-Safe Path Escaping
- New EscapePath() method handles all Windows edge cases:
  * Spaces in paths
  * Backslashes (converted to forward slashes)
  * Long paths (strips \\?\ prefix)
  * Special characters preserved in quotes
  * Embedded quotes escaped properly

### 4. Process Spawning Improvements
- Added working directory setting to prevent path resolution issues
- Maintains existing security (UseShellExecute = false)
- No console window popup (CreateNoWindow = true)

### 5. Hardware Acceleration Verification
- Verified NVIDIA NVENC detection (via ffmpeg -encoders + nvidia-smi)
- Verified AMD AMF detection (via ffmpeg -encoders)
- Verified Intel QuickSync detection (via ffmpeg -encoders)
- GPU memory monitoring working on Windows

### 6. Comprehensive Testing
- Created 11 integration tests for Windows-specific functionality
- Added PowerShell validation script (9 automated checks)
- Zero linter errors, no compilation warnings
- All tests passing

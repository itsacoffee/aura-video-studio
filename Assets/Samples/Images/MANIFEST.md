# Sample Images Manifest

This file describes the sample images that should be included in the Assets/Samples/Images directory.

## Image Specifications

All sample images should be:
- CC0 (Public Domain) or similar permissive license
- High quality (at least 1920x1080 for 16:9, 1080x1920 for 9:16)
- Suitable for video backgrounds
- Royalty-free with commercial use allowed

## Implementation Notes

For the initial implementation, we use programmatically generated solid color images as placeholders. These can be replaced with actual CC0-licensed images from sources like:
- Unsplash (https://unsplash.com/license)
- Pexels (https://www.pexels.com/license/)
- Pixabay (https://pixabay.com/service/license/)

The SampleAssets.cs service will handle loading and serving these images.

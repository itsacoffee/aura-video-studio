# Sample Assets

This directory contains built-in sample assets for immediate testing and demo purposes.

## Directory Structure

- **Images/** - Royalty-free sample images for video generation
- **Audio/** - CC0 licensed background music tracks
- **Templates/** - Demo brief templates for quick start

## Usage

These assets are automatically available in the application and can be used for:
1. Testing video generation without external dependencies
2. Demo and tutorial purposes
3. Quick start templates for new users

## Licensing

All assets in this directory are either:
- CC0 (Public Domain) licensed
- Royalty-free with commercial use allowed
- Created specifically for this project

See individual asset metadata for specific licensing information.

## Adding New Samples

When adding new sample assets:
1. Ensure proper licensing (CC0 or compatible)
2. Update the SampleAssets.cs service to register the new asset
3. Add appropriate metadata and attribution information
4. Test that the asset works in video generation pipeline
